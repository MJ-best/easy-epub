# 실시간 미리보기 기능 구현 완료

## 새로운 기능

### 1. 가로 모드 (Landscape)
- **왼쪽 패널**: 마크다운 편집기
  - 제목, 저자, 본문 입력 필드
  - 템플릿 선택
  - 전자책 생성 버튼

- **오른쪽 패널**: 실시간 EPUB 미리보기
  - 마크다운 내용을 입력하면 즉시 HTML로 변환되어 표시
  - EPUB 2.0 스타일이 적용된 상태로 미리보기
  - 스크롤 가능한 전체 내용 표시

### 2. 세로 모드 (Portrait)
- **단일 패널**: 편집기
  - 모든 입력 필드와 옵션 표시

- **미리보기 버튼**:
  - 본문 내용이 있을 때만 표시
  - 클릭 시 전체 화면 다이얼로그로 미리보기 표시
  - EPUB 리더처럼 렌더링된 결과 확인 가능

## 기술 구현

### OrientationBuilder 사용
```dart
OrientationBuilder(
  builder: (context, orientation) {
    final isLandscape = orientation == Orientation.landscape;

    if (isLandscape) {
      // 왼쪽: 편집기, 오른쪽: 실시간 미리보기
      return Row(children: [editor, preview]);
    } else {
      // 세로: 편집기 + 미리보기 버튼
      return editor;
    }
  },
)
```

### 실시간 마크다운 파싱
- `MarkdownParser.parseToHtml()` 사용
- 사용자가 타이핑하는 즉시 HTML로 변환
- CSS 스타일이 적용된 상태로 렌더링

### EPUB 2.0 스타일 적용
CSS 스타일이 예제 EPUB과 동일하게 적용됨:
- **글꼴**: KoPub Batang (본문), KoPub Dotum (제목)
- **행간**: 1.8em (한국어 서적 표준)
- **정렬**: justify (양쪽 정렬)
- **자간**: -0.02em (한글 최적화)
- **단어 줄바꿈**: keep-all (단어 단위 줄바꿈)

## 업데이트된 파일

### 1. `lib/presentation/create_book/create_book_screen.dart`
**새로운 함수들:**
- `_buildEditorPanel()`: 편집기 패널 구성
- `_buildPreviewPanel()`: 실시간 미리보기 패널 구성
- `_showPreviewDialog()`: 세로 모드 미리보기 다이얼로그
- `_wrapHtmlWithStyles()`: HTML에 EPUB 스타일 적용

**새로운 위젯:**
- `_HtmlPreviewWidget`: HTML을 TextSpan으로 파싱하여 렌더링

### 2. `lib/data/services/epub_generator_service_v2.dart`
**CSS 업데이트:**
- 예제 EPUB의 타이포그래피 스타일 적용
- 한국어 서적에 최적화된 폰트 스택
- 전문적인 북 레이아웃 스타일
- letter-spacing, word-break 등 한글 최적화

## 사용 방법

### 가로 모드에서:
1. 앱을 가로 방향으로 전환
2. 왼쪽에서 마크다운 내용 입력
3. 오른쪽에서 실시간으로 렌더링 결과 확인
4. 타이핑하는 즉시 업데이트됨

### 세로 모드에서:
1. 마크다운 내용 입력
2. "미리보기" 버튼 클릭
3. 전체 화면으로 EPUB 렌더링 결과 확인
4. 닫기 버튼으로 편집 화면 복귀

## 미리보기 기능

### 지원하는 마크다운 문법:
- ✅ 제목 (# ## ###)
- ✅ 단락
- ✅ 굵은 글씨 (**text**)
- ✅ 기울임 (*text*)
- ✅ 코드 (`code`)
- ✅ 순서 없는 목록 (- *)
- ✅ 순서 있는 목록 (1. 2. 3.)
- ✅ 링크 ([text](url))

### 렌더링 스타일:
- **H1**: 1.6em, 굵게, 위아래 여백
- **H2**: 1.3em, 굵게
- **H3**: 1.1em, 굵게
- **본문**: 1em, 1.8 줄간격, 양쪽 정렬
- **목록**: 들여쓰기, 글머리 기호

## 향상된 사용자 경험

1. **즉각적인 피드백**: 입력과 동시에 결과 확인
2. **WYSIWYG**: What You See Is What You Get
3. **모바일 친화적**: 세로/가로 모드 자동 대응
4. **전문적인 렌더링**: 실제 EPUB 리더와 동일한 스타일
5. **편리한 편집**: 미리보기를 보면서 마크다운 수정 가능

## 성능 최적화

- 경량 HTML 파서 사용
- TextSpan 기반 렌더링 (네이티브 성능)
- 불필요한 재렌더링 방지
- 메모리 효율적인 구조
