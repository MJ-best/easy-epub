다음은 **Flutter 기반 EPUB 생성 앱(Product Requirements Document, PRD)** 과 그에 맞는 **UI/UX 설계안**이다. 실제로 실행 가능한 수준으로 구체화되어 있으며, Sigil보다 간결하고 단순한 “템플릿 기반 eBook 자동 생성기”에 초점을 맞추었다.

***

## PRD: “EasyPub – 간편 EPUB 제작기”

### 1. 제품 개요
EasyPub은 일반 사용자가 텍스트 원고를 간단히 입력하거나 붙여넣고, 템플릿을 선택하면 자동으로 아름다운 EPUB 전자책으로 변환해주는 Flutter 기반 앱이다. 기술지식이 없는 사용자도 몇 번의 터치로 EPUB을 생성할 수 있다
### 2. 목표 사용자
- 자기 출판 작가, 블로거, 강의 자료 제작자  
- 출판 편집 입문자 및 교육용 사용자  
- Sigil을 부담스러워하는 일반 사용자  

### 3. 주요 기능
1. **EPUB 생성**
   - 텍스트(또는 Markdown) 입력 후 EPUB 자동 생성  
   - 자동 목차(tv, h1/h2 기준)  
   - 표지 이미지 업로드/선택 기능  
2. **템플릿 시스템**
   - 미리 정의된 3~5가지 eBook 스타일 제공 (소설형, 수필형, 매뉴얼형 등)  
   - 테마색, 글꼴, 여백 설정 옵션  
3. **미리보기 및 내보내기**
   - EPUB 뷰어 내장 (`epub_view` 패키지 연동)  
   - EPUB 파일 저장 및 공유 (로컬 / 클라우드)  
4. **라이브러리 관리**
   - 생성한 eBook 목록  
   - 제목, 표지, 생성일, 수정일 정보 표시  
5. **추가 기능 (향후 확장)**
   - PDF 변환
   - 다국어 지원 (한/영)
   - Google Drive / iCloud 연동  

### 4. 성공 지표
- 1분 내 EPUB 생성 완료율  
- 사용자의 재사용률(월간 2회 이상)  
- 템플릿 전환 UX 이탈률 5% 미만 목표  

### 5. 기술 스택
| 항목 | 기술 |
|------|------|
| Framework | Flutter 3.x |
| EPUB 생성 | epub3 |
| EPUB 미리보기 | epub_view |
| 데이터 저장 | Hive / SQLite |
| Cloud 연동 | Firebase (optional) |
| 디자인 툴 | Figma / Erabook UI Kit[3] |

## UI/UX 설계

### 홈 화면
- **상단 영역:** 앱 이름 “EasyPub”, 설정 아이콘  
- **중앙:** “새 전자책 만들기” 버튼  
- **하단:** 이전에 만든 eBook 목록 (썸네일 + 제목 + 날짜)

### 전자책 생성 화면
- **입력 영역:**  
  - 제목 입력  
  - 저자 이름  
  - 본문 텍스트 입력 (Markdown 지원)  
- **템플릿 선택 슬라이더:**  
  - 카드 형식으로 3~5개 테마 미리보기  
  - 선택 시 실시간 미리보기 갱신  
- **하단 버튼:** 

### 미리보기 화면
- EPUB 뷰어(`epub_view`)로 렌더링  
- 페이지 넘김 애니메이션 (`page_turn` 패키지)  
- 상단 툴바에 “뒤로”, “저장”, “공유” 버튼

### 라이브러리 화면
- Grid/List 전환 가능  
- 각 전자책별 썸네일 + 제목 + 메뉴(공유/삭제)  
- 검색 기능

***

## UX 포인트
1. **“1분 완성” 구조**
   - 입력 – 템플릿 선택 – 저장의 3단계 워크플로우  
2. **실시간 피드백**
   - 템플릿 선택 즉시 미리보기 반영  
   - EPUB 생성 시 로딩 애니메이션 제공  
3. **모바일 최적화**
   - 세로 중심, 손가락 제스처 기반 탐색 UX 적용
1. **Dark/Light Theme 자동 적용**
   - Flutter `ThemeMode.system`로 일관성 유지  

***

이 설계는 **Figma 기반 자동 레이아웃**(Erabook UI Kit 활용)을 적용하면 빠르게 프로토타입 제작이 가능하다
Flutter 환경에서 UI는 `Material 3` 위젯을 기반으로 하되, `page_transition` 애니메이션과 `Container` 그라디언트 배경을 이용해 감성적인 전자책 느낌을 줄 수 있다.
